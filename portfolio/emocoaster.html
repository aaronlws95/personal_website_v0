Emotional Rollercoaster

OVERVIEW

Emotional Rollercoaster is a project developed during IC Hack 2017 (Imperial College London's Annual Hackathon). It is a game where players try to match their emotions with a randomly generated emotion to move a rollercoaster (remote controlled car). How well they match the emotion will determine how far the rollercoaster moves.

The main program was developed in C# and uses Microsoft Cognitive Services. The rollercoaster aspect was developed using an arduino and a hacked remote controlled car. This project was carried out in a team of 5 people where I acted as lead C# developer. You can find out more about the project here! This project won the runner-up prize for "Best Game" at the hackathon.
