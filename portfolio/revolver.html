Revolver

OVERVIEW

Revolver is a simple puzzle game developed as a personal project and exercise in puzzle creation and game development. The aim of the game is to get the key and get to the exit. It is developed in C# MonoGame and currently has 10 playable levels. You check it out on GitHub here or download it here.
