RGB to Depth GAN

OVERVIEW

This project was carried out as part of my final (fourth) year project at Imperial College London in 2017 supervised by Dr. Tae-Kyun Kim. The project title is "Depth to Colour Translation for 3D Hand Pose Estimation From Monocular RGB With Generative Adversarial Networks". Essentially the goal of this project was to harness the capabilities of Generative Adversarial Networks to translate the depth images in the BigHand hand pose estimation dataset to RGB images to assist the training of hand pose estimation networks based on RGB images.

In this project I experimented with Generative Adversarial Networks (GAN) to translate depth images to RGB images. I generated a novel dataset for training this GAN which consists of depth and RGB image pairs. I used a depth camera followed by some image preprocessing to do this. The architecture I used is heavily inspired by pix2pix. I also attempted to modify it and analysed the generated images qualitatively and quantitatively. I also translated the BigHand dataset using the GAN and trained a hand pose esimation network to analyse the results.
