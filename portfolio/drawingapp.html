Drawing Application

OVERVIEW

The Drawing Application was developed during my internship at Materialise Malaysia from July 2016 to September 2016. The application was developed with C++ MFC and implements object-oriented programming.

FEATURES

  • Draw line, circle, rectangle, polyline and polygon
  • Free drawing
  • Calculate properties of line, circle and rectangle
  • Colour outline, fill and unfill
  • Resize outline and shape
  • Save and load with choice of file extension
  • Undo and Redo
  • Delete
  • Regular select, Rectangular multi-select, Shift select
  • Drag and move

SKILLS

  • Application development
  • C++ MFC
  • Object-oriented programming
